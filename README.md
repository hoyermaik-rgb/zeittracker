# Zeittracker PWA ⏱️

Eine private, lokale und hochperformante Zeiterfassungslösung für Desktop und Mobile.

## 🚀 Features

- **Live-Timer:** Echtzeit-Erfassung mit Pause-Funktion.
- **Multi-View:** Schneller Wechsel zwischen Timer, Kalender, Bearbeitungsmodus und Jahresauswertung.
- **Intelligente Statistik:** Automatische Berechnung von Überstunden, Soll-Arbeitszeit und Tagestypen (Arbeit, Urlaub, Krank).
- **PWA Ready:** Kann als App auf dem Homescreen installiert werden (inkl. Icon).
- **Synchronisierung:** Einfacher Datenaustausch zwischen PC und Handy via JSON-Import/Export.
- **CSV & Print:** Export für die Buchhaltung oder zum Ausdrucken.

## 📱 Installation (Mobil)

1. Hoste die `index.html` (z.B. via GitHub Pages).
2. Öffne die URL auf deinem Smartphone.
3. Wähle im Browser-Menü **"Zum Home-Bildschirm hinzufügen"**.

## 💾 Datensicherheit

Alle Daten werden im `LocalStorage` deines Browsers gespeichert. 
- **Kein Server-Backend:** Deine Daten gehören dir.
- **Backup:** Nutze regelmäßig den "Backup"-Button, um deine Daten als JSON zu sichern.

## 🛠️ Anpassung

In der Jahresauswertung kannst du deine **Soll-Stunden** pro Tag und deine **Arbeitstage** (z.B. Mo-Fr) individuell konfigurieren.
